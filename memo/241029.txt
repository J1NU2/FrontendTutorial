0. HTML문서는 HTML을 기본으로 <style>, <javascript> 영역으로 나뉜다.
    JQuery는 자바스크립트로 개발된 일종의 라이브러리로 HTML문서에서는 자바스크립트 영역에서 코딩하면 된다.
1. HTML문서에서는 <div>영역을 잘 나눠야 깔끔하게 구성할 수 있다.
2. 시맨틱 태그를 활용하여 메인화면을 구성한다.
    <header>, <nav>, <article>, <section>, <footer>, <main> 등
3. 태그는 태그명과 속성으로 되어있다.
    태그의 정의와 태그가 갖는 속성을 숙지하여야 한다.
4. 하나의 문서를 Document라고 하고, Document를 구성하는 Object를 
    현재 태그의 부모, 형제, 자식, 자손을 찾아낼 수 있어야 한다.
5. HTML문서를 동적으로 제어하기 위해서는 Object를 선택하고, 선택한 Object에 디자인 적용, 데이터 값 변경 등..
    전체 문서를 새로고침하지 않고, 부분 Object의 내용을 변경할 수 있다. (예) ID중복체크, 좋아요)
6. 위에서 언급한 Object를 선택할 때는 선택자를 선택한다고 한다.
    방법으로는 태그를 선택하는 방법, 태그에 지정된 아이디로 선택하는 방법, 태그에 지정된 클래스로 선택하는 방법 등이 있다.


1. HTML문서의 기본 구조
2. 태그와 속성 이해
3. 태그의 속성 중 아이디는 문서 내에서 유니크해야 한다.
4. 태그의 속성 중 클래스를 유니크하지 않아도 된다.
5. 스타일과 스크립트 태그의 의미와 위치에 따른 로딩 순서
6. 스타일에서의 주석 표시 기호
7. 스크립트에서의 주석 표시 기호
8. 스타일에서의 선택자, 선택하는 방법 3가지(태그, 아이디, 클래스)
9. div로 화면을 분할하는 방법
    display 또는 float 등으로 위치를 배치할 수 있다.


HTML 초기 구성
1. body태그 설정
    1280x1024(width,height), margin/padding
2. 화면분할
    div태그 → 시맨틱태그 : 공간을 분할, 공간의 의미부여
        header : 사이트 메인 제목
        nav : 메뉴
        section : 주요내용
        aside : 서브메뉴(로그인, 배너)
        footer : 개발자
3. display 설정
    block : 한줄차지 + 줄바꿈(세로) + 크기 조절 가능
    inline : 자동으로 영역만큼 차지 + 줄바꿈X(가로) + 크기 지정 불가
    inline-block : inline+block, 자동으로 영역만큼 차지 + 줄바꿈X + 크기 조절 가능
