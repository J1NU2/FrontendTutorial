1. 선언문 : var, let, const
    Scope 구분..
    var : 함수단위 Scope, 중복 선언 및 재할당 가능
    let : 블록단위 Scope, 재할당은 가능하나 중복 선언은 불가능
    const : 상수, 중복 선언 및 재할당 불가능

2. 연산자 : ==, ===
    == : 값만 같으면 true
    === : 자료형까지 같아야 true
